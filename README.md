React-旅遊網站範例練習

手動配置webpack建立react開發環境開發，並使用react-redux架構。

使用 express 撰寫 API 串接 。
 
### Framework

React、React Hooks

### CSS Framework

SCSS、styled-components、react-bootstrap

### libary

Redux、React-Redux、Redux-Thunk、Webpack

### other

axios、react-transition-group、react-router-dom

### npm run start

啟動網站，預設port為9000

### npm run build

專案打包，生成dist資料

### npm run server

打包server.js，用於SSR

### npm run deploy

佈署至github pages

旅遊資料來源 : 東南旅遊、Airbnb
